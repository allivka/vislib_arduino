#

Some implementations of vislib functionality for arduino

## version: 0.0.2

## installation

vislib_arduino can be either manually added to project or installed with platformio package manager by adding "alivka/vislib_arduino" to lib_deps field of platformio.ini file, e.g.:

```ini
[env]
platform = atmelavr
board = megaatmega2560
framework = Arduino
lib_deps =
    alivka/vislib_arduino
    ...
```
